# demopro2023
2023年度のデモプロ用のプログラム<br>
dynamixcelサーボの制御と山彦の前進後進を司る．<br>

dynamixcel_sdk,<br>
dynamixcel_sdk_example,<br>
sound_play<br>
が必須．

# 起動方法
roslaunch demopro2023 harisen_controller.launch //ypspur,soud_play, dynamixcel_sdkが起動<br>
rosrun demopro2023 harisen_controller //harisencontrollerが起動<br>
